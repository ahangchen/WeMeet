# 第一期后小版本（1.0.1）规划
## 私信机制
### 学生端
#### 全局入口
- 所有页面增加消息提示
- 查看会话列表（有更新的会话&&历史所有会话）
- 进入某个会话后查看会话中消息记录（对话形式）
#### 投递入口
- 在自己的投递职位详情页面增加会话入口
- 查看投递的某个职位所属的公司 和 自己 的消息记录
- 可以在消息记录底部增加新消息

### 团队端
#### 全局入口
- 所有页面增加消息提示
- 查看会话列表（有更新的会话&&历史所有会话）
- 进入某个会话后查看会话中消息记录（对话形式）

#### 投递入口
- 在单个已发布职位详情页面增加会话入口
- 查看投递的某个职位的个人 和 自己 的消息记录
- 可以在消息记录底部增加新消息
